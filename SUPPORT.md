# Graffiti Guard Support

Graffiti Guard performs private, offline graffiti detection on iPhone and iPad.

For help, bug reports, or accessibility feedback, [open a GitHub issue](https://github.com/EfficientTools/graffiti-detection-ai-model/issues/new/choose). Include the device model, iOS or iPadOS version, and a description of what happened. Do not attach private photos or sensitive location information.

The detector supports maintenance decisions and should not replace human review. Detection accuracy can vary with lighting, distance, image quality, murals, and unfamiliar graffiti styles.
